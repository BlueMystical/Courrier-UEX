Privacy Policy

// Courrier-UEX Desktop Application
NO personal data is collected or transmitted to the developer. All sensitive data stays on your machine.
01 WHO WE ARE

Courrier-UEX is a free, open-source desktop application developed by BlueMystic and distributed exclusively via GitHub Releases. It is a companion tool for players of Star Citizen, interfacing with the UEX Corp public API to display market and trading data.

The application is not affiliated with Cloud Imperium Games or UEX Corp beyond using their public API.

02 DATA WE DO NOT COLLECT

The developer operates no servers, databases, or analytics services. We never collect, receive, or store:

  • NO Crash reports or telemetry
  • NO Usage statistics or analytics
  • NO IP addresses or device identifiers
  • NO Your Star Citizen username or handle
  • NO Your UEX secret key or any tokens
  • NO Any data beyond what stays on your own device
03 DATA STORED LOCALLY ON YOUR DEVICE

The following data is stored exclusively on your local machine, in the application's user data folder (%APPDATA%\Courrier-UEX\settings.json):

  • LOCAL Your Star Citizen username / handle — used to authenticate with the UEX API
  • LOCAL Your UEX secret key — stored only on your device, never transmitted to the developer
  • LOCAL Application preferences (theme, layout, shortcuts)

This data never leaves your machine except when sent directly to the UEX API (see section 4). You can delete all local data at any time by uninstalling the application or manually deleting the settings file.

04 THIRD-PARTY API — UEX CORP

Courrier-UEX communicates with the UEX Corp API (api.uexcorp.uk) to retrieve market prices, commodity data, and to submit trading data on your behalf.

  • UEX Your username and secret key are sent to UEX servers for authentication
  • UEX Price and inventory data you submit is sent to the UEX API
  • UEX Market data (prices, terminals, commodities) is fetched from UEX servers

The handling of your data by UEX Corp is governed by their own privacy policy. Courrier-UEX has no control over UEX Corp's data practices. We recommend reviewing the UEX Corp website for their terms and privacy information.

05 DONATIONS

Courrier-UEX accepts voluntary donations to support development. Donations are processed through third-party platforms (e.g. Ko-fi, GitHub Sponsors). Any personal information provided during a donation (name, email, payment data) is handled exclusively by those platforms — not by BlueMystic.

Donating does not grant any additional rights or features, and is never required to use the application.

06 CHANGES TO THIS POLICY

If this policy changes in a meaningful way, a new version will be distributed with the next application release. The updated date at the top of this document will reflect the most recent revision. Continued use of the application after an update constitutes acceptance of the revised policy.
